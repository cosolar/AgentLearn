# 示例 04: 研究助手

## 📖 说明

一个能够搜索信息并生成研究报告的智能 Agent。

## 运行

```bash
python examples/04-research-agent/main.py
```

## 功能

- 🔍 搜索互联网信息
- 📝 整理和分析信息
- 📊 生成结构化报告

## 示例

```
研究主题：人工智能的发展趋势

🔍 正在研究...

==================================================
## 人工智能发展趋势报告

### 1. 技术进展
...

### 2. 应用现状
...

### 3. 未来展望
...
==================================================
```
